## Setting up

```
git clone https://github.com/tomnt/drchrono-api-example-django drchrono/hackathon

cd drchrono
pyenv install 2.7.16
pyenv local 2.7.16
pip install virtualenv
python -m virtualenv venv
source venv/bin/activate

cd hackathon

```
